# :joystick: Conway's Game of Life - Game Boy Edition

This is a working version of Conway's Game of Life for the classic Game Boy. Simply fire up your favorite Game Boy emulator and load the `gameoflife.gb` file. All the details of how the Game of Life works can be found [here](https://conwaylife.com/wiki/Conway%27s_Game_of_Life)
